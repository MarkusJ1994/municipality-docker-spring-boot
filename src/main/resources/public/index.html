Municipality column (Mandatory)

Defines which column holds the municipality names. This is 0-indexed, so if the municipality names are in column A on the Excel sheet, this value should be 0

Value column (Mandatory)

Defines which column holds the color deciding value. This is 0-indexed, so if the values are in column B on the Excel sheet, this value should be 1

Mapping column (Optional)

Defines the mapping function for a color. Can be applied to both number and string values. The mapping column is 0-indexed and decides where the program starts to interpret the rules. See table below for guidance

Color code Data type Primary comparison value Operation Secondary comparison value
Column Mapping column Mapping column + 1 Mapping column + 2 Mapping column + 3 Mapping column + 4
Description A number referencing a color, see below color code section for alternatives A datatype for the comparison values, see below datatype section for alternatives A string or number value A comparison operation, see below operation section for alternatives Optional secondary comparison value for intervals. Only valid for number interval comparisons

Example mapping inside Excel sheet

Below mapping example starts at column B. Given the 0-indexed definition, this would yield a mapping column value of 1

(Row) A B C D E F
1 1 NUMBER 25 eq
2 2 NUMBER 25 lt
3 3 NUMBER 25 gt 75
4 5 STRING "Optisk sortering" eq
  • Row 1 maps any row where value is equal to 25 to color code 1 which is green
  • Row 2 maps any row where value is lesser than 25 to color code 2 which is red
  • Row 3 maps any row where value is greater than 25 and lesser than 75 to color code 3 which is yellow
  • Row 4 maps any row where value is equal to "Optisk sortering" to color code 5 which is beige
